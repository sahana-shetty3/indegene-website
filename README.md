Indegene Website Project

This is a simple website created using HTML, CSS, and JavaScript.
It looks like a company website and includes different sections like header, hero, company services, and testimonials.

---

What this project includes

- Header with logo and menu
- Side menu that opens and closes
- Hero section with buttons
- Company section with hover effects
- Testimonial slider
- Events, blog, and contact sections
- Download button

---

Project files

project folder contains:

- index.html → main structure of website
- style.css → design and layout
- script.js → functionality
- images folder → all images used
- files folder → downloadable file

---

How to run the project

1. Download the project folder
2. Open the folder
3. Double click on index.html
4. Website will open in browser

---

How it works

Menu:
Click on MENU to open the side menu
Click close button or any link to close it

Slider:
Testimonials change automatically
You can also click dots or arrows

Navigation:
Click menu items to go to sections
Page scrolls smoothly

Download:
Click download button to download file

---

How to change content

You can edit:

- Text → inside index.html
- Colors and design → inside style.css
- Functionality → inside script.js
- Images → replace inside images folder

---

Important notes

- Keep correct file names
- Keep images in images folder
- Keep download file in files folder
- Make sure script.js is linked at bottom
